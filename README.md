# Claude for Finance and FP&A — Free Resources by Christian Martinez

> Practical guides, prompts, skills, and tools for CFOs, FP&A teams, accountants, and finance professionals who want to use Claude AI to work faster and smarter.

---

## 🦊 About This Repository

**Claude for Finance** is a curated collection of free resources maintained by [Christian Martinez](https://www.youtube.com/@christianmartinezAIforFinance) to help finance professionals get real, practical value from Claude AI.

Whether you're a CFO building AI-powered dashboards, an FP&A analyst automating commentary, or an accountant cleaning ERP data — this repo has something for you.

### What's Inside

| Folder | Description |
|---|---|
| `Claude-skills/` | Ready-to-use Claude Skills for finance workflows (board decks, FP&A commentary, ERP data cleaning, McKinsey-style slides) |
| `Guides/` | PDF guides covering Claude models, live artifacts, prompting for finance |

## 🛠️ Claude Skills for Finance

Pre-built Claude Skills you can load directly into your Claude environment:

| Skill File | Use Case |
|---|---|
| `erp-data-cleaner.skill` | Clean and normalize messy ERP exports |
| `fpa-board-deck.skill` | Generate board-ready FP&A slide narratives |
